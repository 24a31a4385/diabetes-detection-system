Diabetes Detection System

A simple web-based Diabetes Detection System built using HTML and JavaScript.

This project allows users to enter basic health details and checks whether the person is likely diabetic based on simple condition logic.

PROJECT OVERVIEW

The Diabetes Detection System is a beginner-friendly web application that performs a basic diabetes risk check using:

 Age

 Glucose Level

 BMI (Body Mass Index)

Based on user input, the system displays:

 Diabetes Positive

 Diabetes Negative

The prediction is based on a simple rule:

If Glucose > 140 and BMI > 25 → Diabetes Positive

Otherwise → Diabetes Negative
